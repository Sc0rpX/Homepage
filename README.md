# Responsive Portfolio Practice: Ashley Williams

A short practice project focusing on responsive web design using raw HTML and CSS. This project serves as a practical application of CSS Grid, Flexbox, media queries, and fluid typography to create a seamless, adaptable layout across all device sizes (from 1920px desktops down to 320px mobile screens).
